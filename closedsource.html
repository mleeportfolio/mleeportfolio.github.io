Closed Source Projects

As a person who is passionate in the field of software development, creating softwares is one of my hobby and as such I have also developed several open and closed source projects. Please find below some of the closed source projects that I have worked on. 

Financial Report Generator

The financial report generator is developed for a non-profit organization that allows the user to input the general ledger and it will compute the trial balance, income statement and the balance sheet for the reporting year. The output is a formal financial report document in pdf format.

Crypto Miner Manager

The crypto miner manager is an application that utilizes bitcoin price and energy price APIs as well as the hashrate and power consumption of each miners to recommend when and how many the miners should turn on or off during different time of day. The project has won 2nd place in the HYL Summer 2021 Hackathon.

Social Media Curation Software

The Social Media Curation Software automates the curation and posting of content for Twitter, Reddit and Linkedin. A sample of what it does can be found in the YYC Happening twitter page.

Financial Analysis Kit

The Financial Analysis Kit, of FA Kit, is a set of libraries developed in Python to automate fundamental and technical analysis of stocks within the stock market. Using a stock ticker symbol, the FA Kit collects data from Yahoo Finance API and can produce fundamental analysis using Benjamin Formula, Patrioski F-Score and Altman Z Score. The FA Kit can also generate graphs using its functions for pattern recognition. A small portion of the FA Kit's functionality can be found in the website Capital Momentum Dashboard
