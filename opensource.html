BudLib: OpenSource Library Management System

A library management system designed for the Waldorf School that focuses on simplicity and flexibility. The system is designed wih MongoDB as the database, Java Springboot for the API and REACT for the front-end. This system will be available as a client-server solution and can be deployed as a web/cloud based solution as well as a desktop application supporting all major operating systems (Windows/Linux/OSX)

Archive-Org-Batch-Link-Generator

Archive.org is a treasure chest, but sometimes it is hard to extract the download links in bulk from a search of more than 100 results.

What this code does is using selenium to capture all of the links within a collection, go through each of these links and scrape the url of the file extension (All caps) you specified in the code. The links will then be outputted into an excel spreadsheet that you can just use tools such as JDownloader to download all of the files.

Social Club Loaner

This is a DJANGO site with the focus in Social Club asset management. In social clubs, you will often have equipment and assets such as books, board games etc, that can be lend out to its members.

I have noticed in the social clubs that I join, these assets were often kept track using a notebook, or even worse, it isn't being recorded at all.

This website aims to provide the function in keeping track of a social club's assets in a simple interface.
